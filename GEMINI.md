# Gemini CLI — Project Configuration

Read `AGENTS.md` in this project root. It is the source of truth for all
project rules, commands, context files, and boundaries.

When working inside a subdirectory, also read the `AGENTS.md` in that
directory if one exists — it contains module-specific rules.

Use `/memory show` to inspect loaded context. Use `/memory refresh` if
AGENTS.md has been updated during the session.
